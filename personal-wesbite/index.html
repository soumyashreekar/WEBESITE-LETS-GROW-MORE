Features

Application Development

The program enables learners to successfully apply core Java programming languages features & software patterns needed to develop maintainable mobile apps comprised of core Android components, as well as fundamental Java I/O & persistence mechanisms.

  • Create a Virtual Reality app in Android
  • Develop interactive Android App using sensors and touch control
  • C​reate realistic views of 3D objects using lighting, illumination and texture mapping
  • Apply OpenGL ES to draw 3D graphical objects in Android

Data Science

The program will provide with the latest job-ready tools and skills, including open source tools and libraries, Python, databases, SQL, data visualization, data analysis, statistical analysis, predictive modeling.

  • Learn what data science is, the various activities of a data scientist’s job, and methodology to think and work like a data scientist
  • Tools: Jupyter / JupyterLab, GitHub, R Studio
  • Libraries: Pandas, NumPy, Matplotlib, Seaborn, Folium, ipython-sql, Scikit-learn, ScipPy, etc.
  • Projects: Random Album Generator, Predict Housing Prices, Best Classifier Model, Battle of Neighborhoods

Full Stack Web Development

The program covers how to write syntactically correct HTML5 and CSS3, and how to create interactive web experiences.

  • Develop with front-end development languages and tools such as HTML, CSS, JavaScript, React and Bootstrap
  • Program applications using back-end languages and frameworks like Express, Node.js, Python, Django, etc.
  • Deploy and scale applications using Cloud Native methodologies and tools like Containers, Kubernetes, Microservices and Serverless Functions

Machine Learning and Artificial Intelligence

The program will provide with solid theoretical understanding and considerable practice of the main algorithms, uses, and best practices related to Machine Learning

  • Tools: Jupyter Notebooks and Cloud Services (AWS, GCP, Azure, IBM Waatson)
  • Libraries: Pandas, NumPy, Matplotlib, Seaborn, ipython-sql, Scikit-learn, ScipPy, Keras, and TensorFlow.
  • Types: Supervised, Unsupervised, Deep Learning
  • Projects: Sentiment Analysis, Enron Investigation Project, Speech Emotion Recognition Machine Learning Project, Catching Illegal Fishing Project, Online Grocery Recommendation using Collaborative Filtering

I want to sign up for a free 7-day trial, will I be charged?

If you’re new to IT-Sensei, you won’t be charged during your free 7-day trial. So sit back and enjoy! The payment method you’ll provide here will be automatically charged when your trial ends. If you’ve already burned through your free trial within the year, and are re-signing up, you’ll be charged immediately at sign up. After you’re charged, your membership will automatically renew on the same day, either monthly or annually, depending on the plan you’ve selected.

Can I switch between Plans?

Yes, you can change between plans. For our Professional plan customers, please contact us and we’ll be happy to update your plan. For our Personal plan customers, you can switch plans by visiting your Membership page.

Is there any free trial for Professional Plan?

Please contact us and one of our sales gurus will be happy to assist.

Can I cancel anytime?

Yes, you can cancel your membership at any time by visiting your Membership page. Your membership will auto-renew until you cancel, but why would you want to?
